# SettlementProductionFramework
My production framework mod for Fallout 4, serves as a central foundation for my other production mods.

Copies of the mod installed from GitHub will not be supported as this is actively in development, stable releases are published on Nexus.

Nexus Page: https://www.nexusmods.com/fallout4/mods/96014

To understand how to use this in a mod of your own: https://www.nexusmods.com/fallout4/articles/5982

An update to the article covering new features and some guidelines on contributing to the framework will arrive in the future.
Also probably a more technical overview of the framework will be added here.
